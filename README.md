# ⚡ ZRar - بديل WinRAR الحديث والمجاني لـ Windows

<p align="center">
  <img src="ZRar.ico" width="96" alt="ZRar Logo" />
</p>

تطبيق **ZRar** هو بديل عصري، مجاني، ومفتوح المصدر لبرنامج WinRAR و 7-Zip على نظام تشغيل Windows. تم تصميمه بهندسة برمجية متطورة توفر واجهات مستخدم متميزة (Premium HSL Dark/Light Theme) مع ثبات تام للأداء وتجنب الاختناقات وتجميد واجهة الاستخدام تماماً أثناء المهام الكبيرة.

---

## ✨ المميزات الرئيسية (Key Features)

### 🏎️ الأداء والتحكم (Performance & Control)
* **عزل معالجة ضخم (`Process Isolation`)**: يتم عزل محرك الأرشفة والضغط تماماً في عملية نظام مستقلة (`multiprocessing.Process`). الواجهة لا تتجمد وتظل مستجيبة بمعدل 60 إطاراً في الثانية حتى أثناء ضغط ملفات بحجم 10GB تحت استهلاك CPU بنسبة 100%.
* **تحكم صلب وصارم بالمهام (`OS-Level Suspend/Resume`)**: تحكم فوري بالعمليات الجارية من خلال تعليق معالجة خيوط العملية الفرعية واستئنافها صراحة على مستوى نظام التشغيل (باستخدام `psutil`) لتوفير طاقة جهازك وقتما تشاء بنقرة زر واحدة.
* **إلغاء فوري وصحي (`Instant OS Termination`)**: إلغاء العمليات ينهي خيوط المعالج مباشرة دون انتظار، ويحرر ذاكرة الوصول العشوائي (RAM) ويفك قفل الملفات فورياً.

### 📦 صيغة الأرشيف الذاتية الحصرية (`.zrar`)
* **بصمة وتكامل ثنائي (`Dual Compatibility Wrapper`)**: صيغة أرشفة تعتمد على تشفير وضغط 7z فائق السرعة، ملحق في نهايتها (Footer) بيانات ميتا بترميز JSON تحمل بصمة ZRar التعريفية.
* **التوافقية**: يمكن فتح وقراءة ملفات `.zrar` في أي جهاز كأرشيف 7z عادي باستخدام البرامج العالمية (مثل WinRAR و 7-Zip)، ولكن عند فتحها بـ ZRar تظهر بتنسيق وهوية حصرية متميزة!

### 🔍 فهرسة فائقة السرعة وبحث لحظي (`Fast SQLite Caching & Smart Search`)
* **فهرسةSQLite السريعة (`SQLite Index Caching`)**: يحفظ البرنامج هيكل الملفات للأرشيفات التي تم تصفحها مسبقاً في قاعدة بيانات SQLite محلية. عند إعادة فتح الأرشيف يفتح فوراً في أقل من **10 ملي ثانية**.
* **البحث اللحظي المانع للتشنج (`Debounced Live Search`)**: شريط بحث ذكي يقوم بتصفية محتويات المجلدات والأرشيفات فورياً أثناء الكتابة مع تأخير ذكي (Debounce 150ms) لضمان انسيابية الكتابة وحركة الواجهة.

### 👁️ لوحة المعاينة السريعة الجانبية (`Collapsible Preview Pane`)
* لوحة جانبية تفاعلية وقابلة للطي بالكامل. بمجرد اختيار أي ملف، تعرض معلوماته وبيانات الميتا الخاصة به.
* تدعم المعاينة النصية اللحظية للملفات البرمجية والمستندات النصية (أول 1KB)، ومعاينة مصغرة تترجم الصور التوضيحية والصور الفنية (حتى للملفات المضغوطة بالداخل بشكل آمن وخلفي).

### 🔔 تكامل النظام والتنبيهات التشخيصية (`OS Integration & Diagnostics`)
* **تنبيهات ويندوز الرسمية (`Native OS Notifications`)**: إشعارات نظام فورية بدون أي مكتبات خارجية إضافية تخبرك بحالة اكتمال أو فشل الأرشفة حتى لو كان البرنامج مصغراً.
* **سجل الأخطاء التشخيصي الدوار (`Rotating Logs Manager`)**: تتبع مستمر ودقيق للأحداث والأخطاء بملفات نصية دوارة لا تتجاوز 2MB لكل ملف لضمان سلامة جهاز المستخدم.
* **قائمة الزر الأيمن لويندوز (`Context Menu & File Association`)**: ملف تسجيل ريجستري لتفعيل خيارات "الضغط بـ ZRar" و "فك الضغط بـ ZRar" بالنقر الأيمن وربط أيقونة البرنامج بملفات `.zrar` لتفتح تلقائياً بنقرة مزدوجة.

## 🛠️ هيكلية الكود البرمجي (Directory Structure)

```text
e:/بديل win rar/
├── zrar.py                 # نقطة الدخول ومفسر CLI للبرنامج
├── zrar.log                # ملف السجل الدوار النشط
├── integrate_zrar.reg      # ملف تسجيل ويندوز للزر الأيمن والامتدادات
└── src/
    ├── __init__.py
    ├── gui/
    │   ├── __init__.py
    │   ├── main_window.py   # واجهة الشاشة الرئيسية، شريط البحث، واللوحة الجانبية للمعاينة
    │   ├── dialogs.py       # نوافذ حوار المعالجات والخيارات والإعدادات
    │   ├── components.py    # مكونات الـ Breadcrumbs والـ TaskCard وجداول الملفات
    │   └── theme.py         # إعدادات نمط الألوان المطور (Dark Indigo & Cyan)
    └── core/
        ├── __init__.py
        ├── archiver.py      # النواة ومحركات الأرشفة والضغط وفك الضغط
        ├── queue_manager.py # مدير العمليات والعمليات الفرعية Isolated Processes
        ├── cache_manager.py # مدير فهرسة SQLite السريع
        ├── logger.py        # مدير السجل الدوار Rolling Logs
        └── utils.py         # الإشعارات، حساب السرعات والتقدم، وتخمين الوقت
```

## 🚀 التشغيل والتثبيت

### المتطلبات المسبقة
```bash
pip install customtkinter py7zr pyzipper psutil pillow
```

### تشغيل الواجهة
```bash
python zrar.py
```

### وسيطات سطر الأوامر (Command Line Arguments)
* **للضغط مباشرة**: `python zrar.py -c "file1.txt" "file2.jpg"`
* **لفك الضغط مباشرة**: `python zrar.py -x "archive.zip"`
* **لتصفح مجلد معين مباشرة**: `python zrar.py "C:\Users\Username\Downloads"`
